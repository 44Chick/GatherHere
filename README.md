## 🔑 핵심 기능
+ JWT를 이용한 로그인/회원가입 기능
+ 카카오 로그인 서비스 연동
+ 영화, 도서, 음반 데이터 크롤링
+ 감상평 등록 및 삭제
+ 즐겨찾기 저장 및 삭제
